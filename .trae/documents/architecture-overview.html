09 / API Design

API 接口设计

RESTful API + WebSocket 双协议，覆盖市场数据、巨鲸追踪、赛事分析与信号对比四大领域。

GET /api/markets 获取 CS2 活跃市场列表
Query: { sport?, event_tier?, sort_by?: "volume" | "change" | "liquidity", page? } Response: { markets: Array<{ condition_id, question, slug, tokens, volume, liquidity, change_24h, teams, match_time }>, total, page }
GET /api/whales 获取大户排行
Query: { market_id?, sort_by?: "volume" | "pnl" | "suspicion", limit? } Response: Array<{ address, label?, total_volume, total_pnl, suspicion_score, active_positions, last_active }>
GET /api/signals/:marketId 获取预测信号与多源对比
Response: { polymarket_price, model_prediction, community_consensus, deviation, arbitrage_opportunities: Array<{ type, profit_pct, description }>, confidence, factors: Array<{ name, weight, direction: "bullish" | "bearish" | "neutral" }> }
WS /ws 实时数据推送
// 客户端订阅 { type: "subscribe", channels: ["price", "trades", "book", "alerts"], markets?: [...] } // 服务端推送 { type: "price_update" | "trade" | "book_update" | "whale_alert" | "anomaly", ...payload }
